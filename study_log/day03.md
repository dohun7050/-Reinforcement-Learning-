# 다중 슬롯 머신 (Multi-Armed Bandit)

## 개념
- 여러 선택지(슬롯머신) 중 하나를 반복적으로 선택하여 최대 보상을 얻는 문제
- 각 슬롯머신은 확률적 보상 분포를 가짐
- 선택할 때마다 다른 보상이 나올 수 있음

---

## 탐색 vs 활용

### 탐색 (Exploration)
- 새로운 슬롯머신을 시도
- 장점: 더 높은 보상을 발견할 가능성
- 단점: 현재보다 낮은 보상을 받을 수 있음

### 활용 (Exploitation)
- 현재까지 가장 좋은 선택을 반복
- 장점: 안정적인 보상 확보
- 단점: 더 좋은 선택지를 놓칠 수 있음

---

## 슬롯 머신 가치 추정 방법

### 1. 그리디 알고리즘
- 현재 평균 보상이 가장 높은 팔 선택

#### 한계
- 초기 선택이 잘못되면 최적해를 놓칠 수 있음

---

### 2. 입실론 그리디 (Epsilon-Greedy)
- 확률 ε로 무작위 선택 (탐색)
- 확률 (1 - ε)로 현재 최적 선택 (활용)

#### 특징
- 탐색과 활용의 균형

#### 장점
- 최적해에 접근 가능

#### 단점
- 탐색 비용 발생
- ε 값 설정이 어려움

---

### 3. 신뢰상한행동선택 (UCB, Upper Confidence Bound)
- 평균 보상이 높고, 덜 선택된 팔을 우선 선택

#### 특징
- 탐색 부족 문제 해결
- 불확실성이 큰 선택지를 우선 고려

---

### 4. 톰슨 샘플링 (Thompson Sampling)
- 각 팔에 대해 확률 분포 설정
- 샘플링하여 가장 큰 값을 가진 팔 선택
- 선택 결과를 기반으로 분포 업데이트 반복

#### 특징
- 확률 기반 접근
- 일반적으로 좋은 성능

---

## 긍정적 초깃값 (Optimistic Initial Values)
- 초기 값을 높게 설정하여 자연스럽게 탐색 유도

---

## 점진적 구현 (Incremental Implementation)

### 문제점
- 모든 과거 데이터를 저장하면:
  - 메모리 낭비
  - 계산 시간 증가

### 해결
- 이전 평균을 이용해 새로운 값으로 업데이트

---

## 점진적 평균식

### 기본 식
Qₙ = Qₙ₋₁ + (1/n)(Rₙ - Qₙ₋₁)

또는

Q = Q + (1/n)(r - Q)

- Q: 현재 추정값
- r: 현재 보상
- n: 선택 횟수

#### 특징
- n이 커질수록 변화량 감소

---

## 지수 이동 평균 (Exponential Moving Average)

Qₙ = Qₙ₋₁ + α(Rₙ - Qₙ₋₁)

- α: 학습률 (0 < α ≤ 1)

#### 특징
- 최근 데이터에 더 큰 가중치

---

## 장점
- 메모리 효율성
- 계산 효율성
- 실시간 업데이트 가능

---

## 핵심 정리
- 탐색 vs 활용의 균형이 핵심
- 다양한 알고리즘으로 이를 해결
- 점진적 업데이트로 효율적인 계산 가능
