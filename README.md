# -Reinforcement-Learning-

## 1. 프로젝트 목표
본 프로젝트는 강화학습(Reinforcement Learning)을 기초부터 학습하고,
이를 바탕으로 연구 과제 수행 및 포트폴리오 구축을 목표로 한다.
단순 이론 학습이 아니라, 직접 알고리즘을 구현하고 실험 결과를 기록하는 것을 중심으로 진행한다.

## 2.개발 환경
Language: Python
IDE: VS Code
Notebook: JupyterLab

## 3.학습 방향
강화학습을 다음과 같은 순서로 학습한다.
강화학습 기본 개념 이해
Q-learning 구현
SARSA 알고리즘 비교
Deep Q Network (DQN) 구현
실험 및 성능 개선

## 4.핵심 개념 정리
강화학습은 에이전트(Agent)가 환경(Environment)과 상호작용하며
보상(Reward)을 최대화하는 방향으로 학습하는 방법이다.
